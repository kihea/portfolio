# Divine Ipseity — Web (marketing site)

The master‑brand marketing surface. Homepage, manifesto/about, essay archive, subscribe.

This is the **public face** for the umbrella brand: it sets the tone, hosts the manifesto, lists recent WithDepth and Systematic Sins entries, and feeds new readers into the newsletter / Faction of Truth funnel.

## ⚠ This kit is a **first realization**, not a recreation

No codebase, Figma file, or screenshots were available for Divine Ipseity at design‑system time. This kit is designed *from* the strategy memo, in service of *future* implementation. Treat it as a binding visual contract for whoever builds the real site.

## Components in this kit

- `Header.jsx` — sub‑brand stripe + italic wordmark + sans nav
- `HeroManifesto.jsx` — homepage hero, with the master positioning statement
- `EssayList.jsx` — recent‑essays index with eyebrow / title / lede
- `SubscribeBlock.jsx` — newsletter capture, single field
- `Footer.jsx` — wordmark bottom‑left, sub‑brand list, hairline rule

## Screens

`index.html` walks through:
1. Home (hero + recent essays + subscribe)
2. Manifesto (the "why anonymous" page)
3. Essays archive (chronological, by sub‑brand)

Click between them with the inline nav.
