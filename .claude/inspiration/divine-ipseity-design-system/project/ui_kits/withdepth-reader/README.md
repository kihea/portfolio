# WithDepth — long-form essay reader

The Sage arm's reading surface. Where 4,000–10,000-word essays live, with footnotes, citations, drop caps, and a generous 62ch measure.

## ⚠ This kit is a **first realization**, not a recreation
No existing codebase or screenshots were available. Built from the strategy memo's voice and visual rules.

## Components
- `EssayHeader.jsx` — eyebrow + title + lede + meta
- `EssayBody.jsx` — prose, drop cap, pull-quote, footnote refs
- `Footnotes.jsx` — IBM Plex Mono ledger of citations at the foot of the essay
- `ReaderChrome.jsx` — minimal nav + progress rule
